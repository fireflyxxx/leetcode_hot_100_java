## 53. 最大子数组和（Maximum Subarray）· Medium

🔗 https://leetcode.cn/problems/maximum-subarray/

### 题目大意
找出和最大的**连续**子数组，返回其和。
例：`[-2,1,-3,4,-1,2,1,-5,4]` → 子数组 `[4,-1,2,1]`，和 `6`。

### 思路
**动态规划（Kadane 算法）**。定义 `cur` = "以当前元素结尾的最大子数组和"。
对每个元素，要么把它接在前面那段后面（`cur + nums[i]`），要么自己另起一段（`nums[i]`）——取较大者。当前面那段和为负时，接上只会拖累，不如重开。
全程维护一个全局最大值 `max`。

转移：`cur = max(nums[i], cur + nums[i])`，答案 `max = max(max, cur)`。

### 复杂度
- 时间 O(n)，空间 O(1)。

### 代码
```java
class Solution {
    public int maxSubArray(int[] nums) {
        int max = nums[0];  // 全局最大子数组和
        int cur = nums[0];  // 以当前元素结尾的最大子数组和

        for (int i = 1; i < nums.length; i++) {
            // 要么把前面那段接上，要么从当前元素重新开始
            cur = Math.max(nums[i], cur + nums[i]);
            max = Math.max(max, cur); // 更新全局答案
        }
        return max;
    }
}
```

### 小结 / 语法点
- Kadane 是"线性 DP + 滚动变量"的最经典例子：`dp[i]` 只依赖 `dp[i-1]`，所以用一个变量即可。
- 关键判断："前缀和为负就丢弃"。
- 初始值取 `nums[0]`，循环从 1 开始（数组非空）。

## 56. 合并区间（Merge Intervals）· Medium

🔗 https://leetcode.cn/problems/merge-intervals/

### 题目大意
合并所有重叠的区间。
例：`[[1,3],[2,6],[8,10],[15,18]]` → `[[1,6],[8,10],[15,18]]`。

### 思路
**按左端点排序 + 一次扫描合并**：
1. 先按区间起点升序排序，保证可能重叠的区间相邻。
2. 维护结果列表，遍历每个区间：
   - 若它的起点 **大于** 结果中最后一个区间的终点 → 不重叠，直接加入。
   - 否则重叠 → 把最后一个区间的终点更新为两者终点的较大值（合并）。

### 复杂度
- 时间 O(n log n)（排序主导），空间 O(n)。

### 代码
```java
class Solution {
    public int[][] merge(int[][] intervals) {
        // 按区间起点升序排序（Integer.compare 防溢出，比 a[0]-b[0] 更稳）
        Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0]));

        List<int[]> res = new ArrayList<>();
        for (int[] cur : intervals) {
            // 结果为空，或当前区间与上一个不重叠（起点 > 上一个的终点）→ 新开一个
            if (res.isEmpty() || res.get(res.size() - 1)[1] < cur[0]) {
                res.add(cur);
            } else {
                // 重叠：合并，更新上一个区间的终点为较大值
                int[] last = res.get(res.size() - 1);
                last[1] = Math.max(last[1], cur[1]);
            }
        }
        // List<int[]> 转 int[][]
        return res.toArray(new int[res.size()][]);
    }
}
```

### 小结 / 语法点
- **区间题第一反应：按端点排序**。本题按左端点，区间调度题常按右端点。
- 比较器对 `int[]` 用 `Arrays.sort(arr, (a,b)->...)`，按某列排序。
- `List<int[]>` 转二维数组：`res.toArray(new int[res.size()][])`。

## 189. 轮转数组（Rotate Array）· Medium

🔗 https://leetcode.cn/problems/rotate-array/

### 题目大意
把数组元素整体向右轮转 `k` 个位置（要求原地，O(1) 额外空间）。
例：`[1,2,3,4,5,6,7], k=3` → `[5,6,7,1,2,3,4]`。

### 思路
**三次反转**（经典技巧）：
- 先整体反转：`[7,6,5,4,3,2,1]`。
- 再反转前 `k` 个：`[5,6,7|4,3,2,1]`。
- 再反转后 `n-k` 个：`[5,6,7|1,2,3,4]`。完成。

原理：向右转 k 位，等价于把后 k 个搬到前面。整体反转把后 k 个挪到了前面（但内部逆序），再各自反转一次纠正内部顺序。
注意 `k` 可能大于 `n`，先 `k %= n`。

### 复杂度
- 时间 O(n)，空间 O(1)。

### 代码
```java
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k %= n;                  // k 可能超过 n，取模

        reverse(nums, 0, n - 1); // 整体反转
        reverse(nums, 0, k - 1); // 反转前 k 个
        reverse(nums, k, n - 1); // 反转后 n-k 个
    }

    // 反转 a[i..j]（闭区间）
    private void reverse(int[] a, int i, int j) {
        while (i < j) {
            int t = a[i]; a[i] = a[j]; a[j] = t;
            i++;
            j--;
        }
    }
}
```

### 小结 / 语法点
- **三次反转**是数组循环移位的标准原地解法，务必记住。
- 一定要先 `k %= n`，否则 `reverse(nums, 0, k-1)` 会越界。
- 把 `reverse` 抽成私有辅助方法，代码更清晰。

## 238. 除自身以外数组的乘积（Product of Array Except Self）· Medium

🔗 https://leetcode.cn/problems/product-of-array-except-self/

### 题目大意
返回数组 `ans`，其中 `ans[i]` 等于 `nums` 中**除 `nums[i]` 外**所有元素的乘积。要求 **不能用除法**，O(n)。
例：`[1,2,3,4]` → `[24,12,8,6]`。

### 思路
**前缀积 × 后缀积**。`ans[i]` = （i 左边所有数的积）×（i 右边所有数的积）。
- 第一趟从左到右，让 `ans[i]` 先存"左边所有数的乘积"。
- 第二趟从右到左，用一个变量 `right` 累计"右边所有数的乘积"，乘到 `ans[i]` 上。

这样只用了输出数组本身 + 一个变量，额外空间 O(1)。

### 复杂度
- 时间 O(n)，额外空间 O(1)（输出数组不计）。

### 代码
```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] ans = new int[n];

        // 第一趟：ans[i] = nums[0]*...*nums[i-1]（i 左边的乘积）
        ans[0] = 1;                       // 最左边没有左侧元素，积为 1
        for (int i = 1; i < n; i++) {
            ans[i] = ans[i - 1] * nums[i - 1];
        }

        // 第二趟：从右往左乘上“右边的乘积”
        int right = 1;                    // 累计 i 右边的乘积
        for (int i = n - 1; i >= 0; i--) {
            ans[i] *= right;              // 左积 × 右积
            right *= nums[i];             // 更新右积，纳入当前元素
        }
        return ans;
    }
}
```

### 小结 / 语法点
- "不能用除法 + 除自身外乘积" → 条件反射想到**前缀积/后缀积**。
- 用输出数组存前缀积、用单变量滚动后缀积，是空间优化的常见手法。

## 41. 缺失的第一个正数（First Missing Positive）· Hard

🔗 https://leetcode.cn/problems/first-missing-positive/

### 题目大意
给未排序数组，找出**没有出现的最小正整数**。要求 **O(n) 时间 + O(1) 额外空间**。
例：`[3,4,-1,1]` → `2`；`[7,8,9,11,12]` → `1`。

### 思路
**原地哈希（把数组当哈希表用）**。长度为 `n` 的数组，答案一定在 `[1, n+1]` 范围内。
核心：把数值 `v`（当 `1 ≤ v ≤ n`）放到下标 `v-1` 的位置上——让"值 v 待在它该在的家 v-1"。
- 遍历，对每个位置不断交换：若 `nums[i]` 在 `[1,n]` 且没在正确的家，就把它换到家里。用 `while` 因为换来的新值可能也需要继续归位。
- 归位后再扫一遍，第一个"下标 i 处的值不是 i+1"的位置，`i+1` 就是答案；若全部就位，答案是 `n+1`。

### 复杂度
- 时间 O(n)：每个数最多被换到正确位置一次，总交换 O(n)。
- 空间 O(1)：原地。

### 代码
```java
class Solution {
    public int firstMissingPositive(int[] nums) {
        int n = nums.length;

        for (int i = 0; i < n; i++) {
            // 把 nums[i] 放到它应在的位置 nums[i]-1，直到无法继续归位
            // 条件：值在 [1,n] 内，且目标位置上还不是这个值（避免重复值死循环）
            while (nums[i] >= 1 && nums[i] <= n && nums[nums[i] - 1] != nums[i]) {
                int target = nums[i] - 1;      // 用临时变量，避免交换中下标变化导致 bug
                int tmp = nums[target];
                nums[target] = nums[i];
                nums[i] = tmp;
            }
        }

        // 找第一个“位置 i 上不是 i+1”的地方
        for (int i = 0; i < n; i++) {
            if (nums[i] != i + 1) return i + 1;
        }
        return n + 1;                          // 1..n 全部就位，缺的是 n+1
    }
}
```

### 小结 / 语法点
- "**O(1) 空间找缺失/重复的数**" → 想到**原地哈希**（把下标当 key）。
- 交换时务必先把 `nums[i]-1` 存进临时变量 `target`，否则交换后 `nums[i]` 变了会算错下标。
- `while` 而非 `if`：换进来的新值可能还要继续归位。
