# 11 · 回溯（8 题）

> 回溯 = 在决策树上 **做选择 → 递归 → 撤销选择**。求"所有方案"（排列/组合/子集/切割/棋盘）就用它。
> 题目：46 全排列 · 78 子集 · 17 电话号码字母组合 · 39 组合总和 · 22 括号生成 · 79 单词搜索 · 131 分割回文串 · 51 N皇后

> **通用模板**
> ```java
> void backtrack(参数, List<T> path) {
>     if (满足终止条件) { res.add(new ArrayList<>(path)); return; } // 存副本！
>     for (选择 : 当前层所有选择) {
>         if (剪枝) continue;
>         path.add(选择);          // 做选择
>         backtrack(下一层, path); // 递归
>         path.remove(末尾);       // 撤销选择（回溯）
>     }
> }
> ```
> **三种典型的区别**：组合/子集用 `start` 控制不回头；排列用 `used[]` 标记；去重先排序再 `if(i>start && a[i]==a[i-1]) continue`。

## 46. 全排列（Permutations）· Medium

🔗 https://leetcode.cn/problems/permutations/

### 题目大意
返回数组（无重复数字）的所有全排列。

### 思路
**排列型回溯**。每个位置可以选任何"还没用过"的数，用 `boolean[] used` 记录哪些已在路径中。每层都从下标 0 开始尝试（和组合不同，排列要回头取）。路径长度等于数组长度时收集一个排列。

### 复杂度
- 时间 O(n·n!)，空间 O(n)。

### 代码
```java
class Solution {
    public List<List<Integer>> permute(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        boolean[] used = new boolean[nums.length];
        backtrack(nums, new ArrayList<>(), used, res);
        return res;
    }
    private void backtrack(int[] nums, List<Integer> path, boolean[] used, List<List<Integer>> res) {
        if (path.size() == nums.length) {       // 选满 n 个 → 一个排列
            res.add(new ArrayList<>(path));
            return;
        }
        for (int i = 0; i < nums.length; i++) {
            if (used[i]) continue;              // 跳过已用过的
            used[i] = true;                     // 做选择
            path.add(nums[i]);
            backtrack(nums, path, used, res);   // 递归下一层
            path.remove(path.size() - 1);       // 撤销
            used[i] = false;
        }
    }
}
```

### 小结 / 语法点
- 排列用 `used[]` 标记；每层从 0 开始（要回头取）。
- 终止时 `new ArrayList<>(path)` 存副本，否则后续修改会污染已存结果。

## 78. 子集（Subsets）· Medium

🔗 https://leetcode.cn/problems/subsets/

### 题目大意
返回数组（无重复元素）的所有子集（幂集）。

### 思路
**组合型回溯**。决策树的**每一个节点**都是一个子集（不是只有叶子）。用 `start` 控制"只能往后取"，避免重复子集。进入递归前先把当前 path 收集进结果。

### 复杂度
- 时间 O(n·2ⁿ)，空间 O(n)。

### 代码
```java
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        backtrack(nums, 0, new ArrayList<>(), res);
        return res;
    }
    private void backtrack(int[] nums, int start, List<Integer> path, List<List<Integer>> res) {
        res.add(new ArrayList<>(path));          // 每个节点都是一个子集
        for (int i = start; i < nums.length; i++) {
            path.add(nums[i]);                   // 选 nums[i]
            backtrack(nums, i + 1, path, res);   // 从 i+1 往后，不回头
            path.remove(path.size() - 1);        // 撤销
        }
    }
}
```

### 小结 / 语法点
- 子集/组合用 `start` 参数，递归传 `i + 1` 保证不重复、不回头。
- 与全排列对比：子集在**每个节点**收集，排列只在**叶子**收集。

## 17. 电话号码的字母组合（Letter Combinations）· Medium

🔗 https://leetcode.cn/problems/letter-combinations-of-a-phone-number/

### 题目大意
给数字串（2-9），返回它能表示的所有字母组合（电话九宫格）。

### 思路
**多叉回溯**。第 `idx` 个数字对应一组字母，逐位枚举：选当前数字的某个字母 → 递归处理下一位数字 → 撤销。用映射数组 `map[digit] = 字母串`。

### 复杂度
- 时间 O(4ⁿ·n)，空间 O(n)。

### 代码
```java
class Solution {
    // 下标 = 数字，值 = 对应字母
    private String[] map = {"", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};

    public List<String> letterCombinations(String digits) {
        List<String> res = new ArrayList<>();
        if (digits == null || digits.length() == 0) return res; // 空串特判
        backtrack(digits, 0, new StringBuilder(), res);
        return res;
    }
    private void backtrack(String digits, int idx, StringBuilder sb, List<String> res) {
        if (idx == digits.length()) {            // 每位都选了一个字母
            res.add(sb.toString());
            return;
        }
        String letters = map[digits.charAt(idx) - '0']; // 当前数字对应的字母
        for (char c : letters.toCharArray()) {
            sb.append(c);                        // 选字母
            backtrack(digits, idx + 1, sb, res); // 处理下一位数字
            sb.deleteCharAt(sb.length() - 1);    // 撤销
        }
    }
}
```

### 小结 / 语法点
- 不同层的选择来自不同集合（不同数字的字母表），用 `idx` 标记进行到第几位。
- 用 `StringBuilder` 拼接，`deleteCharAt(length-1)` 撤销末位。
- 别忘了**空输入返回空列表**。

## 39. 组合总和（Combination Sum）· Medium

🔗 https://leetcode.cn/problems/combination-sum/

### 题目大意
给无重复正整数数组 `candidates` 和 `target`，找出所有和为 target 的组合，**同一个数可重复选取**。
例：`candidates=[2,3,6,7], target=7` → `[[2,2,3],[7]]`。

### 思路
**组合型回溯 + 可重复**。和子集类似用 `start`，但因为可重复选同一个数，递归时传的是 `i` 而不是 `i+1`。先排序便于剪枝：当 `candidates[i] > remain` 时，后面更大的都不用看了，`break`。

### 复杂度
- 时间与解的数量相关（指数级），空间 O(target)。

### 代码
```java
class Solution {
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(candidates);                 // 排序，便于剪枝
        backtrack(candidates, target, 0, new ArrayList<>(), res);
        return res;
    }
    private void backtrack(int[] cand, int remain, int start, List<Integer> path, List<List<Integer>> res) {
        if (remain == 0) {                       // 凑到目标
            res.add(new ArrayList<>(path));
            return;
        }
        for (int i = start; i < cand.length; i++) {
            if (cand[i] > remain) break;         // 排序后剪枝：当前太大，后面更大，停
            path.add(cand[i]);
            // 传 i 而非 i+1：允许重复使用 cand[i]
            backtrack(cand, remain - cand[i], i, path, res);
            path.remove(path.size() - 1);
        }
    }
}
```

### 小结 / 语法点
- **可重复选** → 递归传 `i`；**不可重复** → 传 `i+1`。
- 排序 + `if (cand[i] > remain) break` 是关键剪枝。
- 用 `remain` 递减比每次求和更高效。

## 22. 括号生成（Generate Parentheses）· Medium

🔗 https://leetcode.cn/problems/generate-parentheses/

### 题目大意
给 `n` 对括号，生成所有**有效**的括号组合。

### 思路
**带约束的回溯**。维护已用的左括号数 `open`、右括号数 `close`：
- 只要 `open < n` 就能加左括号。
- 只要 `close < open`（右不能超过左，否则非法）就能加右括号。
长度达到 `2n` 时收集。这两个约束天然保证了只生成合法括号串。

### 复杂度
- 时间 O(4ⁿ/√n)（卡特兰数级），空间 O(n)。

### 代码
```java
class Solution {
    public List<String> generateParenthesis(int n) {
        List<String> res = new ArrayList<>();
        backtrack(n, 0, 0, new StringBuilder(), res);
        return res;
    }
    private void backtrack(int n, int open, int close, StringBuilder sb, List<String> res) {
        if (sb.length() == 2 * n) {              // 用满 2n 个字符
            res.add(sb.toString());
            return;
        }
        if (open < n) {                          // 还能放左括号
            sb.append('(');
            backtrack(n, open + 1, close, sb, res);
            sb.deleteCharAt(sb.length() - 1);
        }
        if (close < open) {                      // 右括号不能多于左括号
            sb.append(')');
            backtrack(n, open, close + 1, sb, res);
            sb.deleteCharAt(sb.length() - 1);
        }
    }
}
```

### 小结 / 语法点
- 用 **约束剪枝**（`open<n`、`close<open`）直接只生成合法解，避免先全生成再过滤。
- 把"合法性"编码进选择条件，是回溯优化的常见手法。

## 79. 单词搜索（Word Search）· Medium

🔗 https://leetcode.cn/problems/word-search/

### 题目大意
在字符网格中判断是否存在一条相邻路径拼出 `word`（同一格不能重复用）。

### 思路
**网格 DFS 回溯**。从每个格子尝试作为起点 DFS：当前格匹配 `word[k]` 才继续，匹配到最后一个字符即成功。访问时把格子临时改成 `'#'` 标记已用，回溯时恢复（避免同一格重复使用）。

### 复杂度
- 时间 O(m·n·4^L)，L 为 word 长度。空间 O(L)。

### 代码
```java
class Solution {
    public boolean exist(char[][] board, String word) {
        int m = board.length, n = board[0].length;
        for (int i = 0; i < m; i++)
            for (int j = 0; j < n; j++)
                if (dfs(board, word, i, j, 0)) return true;  // 任一起点成功即可
        return false;
    }
    private boolean dfs(char[][] board, String word, int i, int j, int k) {
        // 越界 或 当前格不等于 word 的第 k 个字符 → 失败
        if (i < 0 || i >= board.length || j < 0 || j >= board[0].length
                || board[i][j] != word.charAt(k))
            return false;
        if (k == word.length() - 1) return true;  // 匹配到最后一个字符 → 成功

        char tmp = board[i][j];
        board[i][j] = '#';                        // 标记已访问
        boolean found = dfs(board, word, i + 1, j, k + 1)
                     || dfs(board, word, i - 1, j, k + 1)
                     || dfs(board, word, i, j + 1, k + 1)
                     || dfs(board, word, i, j - 1, k + 1);
        board[i][j] = tmp;                        // 回溯恢复

        return found;
    }
}
```

### 小结 / 语法点
- 网格回溯的"访问标记 + 撤销"：用 `'#'` 占位，递归后恢复原字符。
- 短路 `||` 让任一方向成功就提前返回。

## 131. 分割回文串（Palindrome Partitioning）· Medium

🔗 https://leetcode.cn/problems/palindrome-partitioning/

### 题目大意
把字符串切割成若干子串，使**每个子串都是回文**，返回所有切割方案。

### 思路
**切割型回溯**。`start` 表示当前从哪里开始切。枚举切割终点 `end`，若 `s[start..end]` 是回文，就切下这一段、递归处理 `end+1` 之后，再撤销。`start == 长度` 时说明切完一种方案。

### 复杂度
- 时间 O(n·2ⁿ)，空间 O(n)。

### 代码
```java
class Solution {
    public List<List<String>> partition(String s) {
        List<List<String>> res = new ArrayList<>();
        backtrack(s, 0, new ArrayList<>(), res);
        return res;
    }
    private void backtrack(String s, int start, List<String> path, List<List<String>> res) {
        if (start == s.length()) {               // 切到末尾，得到一种方案
            res.add(new ArrayList<>(path));
            return;
        }
        for (int end = start; end < s.length(); end++) {
            if (isPalindrome(s, start, end)) {   // 只在当前段是回文时才切
                path.add(s.substring(start, end + 1));
                backtrack(s, end + 1, path, res);
                path.remove(path.size() - 1);    // 撤销
            }
        }
    }
    private boolean isPalindrome(String s, int l, int r) {
        while (l < r) {
            if (s.charAt(l++) != s.charAt(r--)) return false;
        }
        return true;
    }
}
```

### 小结 / 语法点
- **切割问题** = 用 `start` 当切割起点，枚举 `end` 决定每段长度。
- `substring(start, end + 1)` 左闭右开取 `[start, end]`。

## 51. N 皇后（N-Queens）· Hard

🔗 https://leetcode.cn/problems/n-queens/

### 题目大意
在 `n×n` 棋盘放 `n` 个皇后，使其两两不能互相攻击（不同行、不同列、不同对角线），返回所有摆法。

### 思路
**逐行回溯**。每行放一个皇后，枚举该行每一列，检查"同列、左上斜、右上斜"是否已有皇后（只需检查上方已放的行）。合法就放下，递归下一行；放满 n 行就记录方案。

### 复杂度
- 时间 O(n!)，空间 O(n²)（棋盘）。

### 代码
```java
class Solution {
    public List<List<String>> solveNQueens(int n) {
        List<List<String>> res = new ArrayList<>();
        char[][] board = new char[n][n];
        for (char[] row : board) Arrays.fill(row, '.'); // 初始化空棋盘
        backtrack(board, 0, res);
        return res;
    }
    private void backtrack(char[][] board, int row, List<List<String>> res) {
        int n = board.length;
        if (row == n) {                          // 放满 n 行
            res.add(build(board));
            return;
        }
        for (int col = 0; col < n; col++) {
            if (isValid(board, row, col)) {
                board[row][col] = 'Q';           // 放皇后
                backtrack(board, row + 1, res);  // 下一行
                board[row][col] = '.';           // 撤销
            }
        }
    }
    // 只需检查上方：同列、左上对角线、右上对角线
    private boolean isValid(char[][] board, int row, int col) {
        int n = board.length;
        for (int i = 0; i < row; i++)
            if (board[i][col] == 'Q') return false;                 // 同列
        for (int i = row - 1, j = col - 1; i >= 0 && j >= 0; i--, j--)
            if (board[i][j] == 'Q') return false;                   // 左上斜
        for (int i = row - 1, j = col + 1; i >= 0 && j < n; i--, j++)
            if (board[i][j] == 'Q') return false;                   // 右上斜
        return true;
    }
    private List<String> build(char[][] board) {
        List<String> list = new ArrayList<>();
        for (char[] row : board) list.add(new String(row));         // char[] → String
        return list;
    }
}
```

### 小结 / 语法点
- 逐行放置天然保证"不同行"；只需检查同列和两条上对角线。
- `Arrays.fill(row, '.')` 初始化棋盘；`new String(row)` 把一行字符数组转字符串。
- 进阶可用三个 `boolean[]`（列、两对角线）把 `isValid` 优化到 O(1)。
