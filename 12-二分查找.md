# 12 · 二分查找（6 题）

> 二分的灵魂是**区间不变量**：想清楚 `[l, r]` 是闭还是半开，循环条件、`mid` 归属、边界移动要配套。
> 题目：35 搜索插入位置 · 74 搜索二维矩阵 · 34 查找首末位置 · 33 搜索旋转排序数组 · 153 旋转数组最小值 · 4 两个正序数组中位数

> **两套最常用模板**
> ```java
> // 模板 A：闭区间 [l, r]，找具体值
> int l = 0, r = n - 1;
> while (l <= r) {
>     int mid = l + (r - l) / 2;
>     if (nums[mid] == target) return mid;
>     else if (nums[mid] < target) l = mid + 1;
>     else r = mid - 1;
> }
> return -1;
>
> // 模板 B：半开区间 [l, r)，找“第一个 >= target”的位置（lower_bound）
> int l = 0, r = n;
> while (l < r) {
>     int mid = l + (r - l) / 2;
>     if (nums[mid] >= target) r = mid;   // 满足，收右边界
>     else l = mid + 1;
> }
> return l;   // 可能等于 n（都比 target 小）
> ```

## 35. 搜索插入位置（Search Insert Position）· Easy

🔗 https://leetcode.cn/problems/search-insert-position/

### 题目大意
在升序数组中找 target；存在返回其下标，不存在返回它**应插入的位置**（保持有序）。

### 思路
就是求"**第一个 ≥ target 的位置**"（lower_bound，模板 B）。若 target 存在，返回它的下标；不存在，返回第一个比它大的位置，正好是插入点。

### 复杂度
- 时间 O(log n)，空间 O(1)。

### 代码
```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        int l = 0, r = nums.length;          // 半开区间 [l, r)，r 取 n
        while (l < r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] >= target) r = mid;   // mid 满足，往左继续找更小下标
            else l = mid + 1;                   // mid 偏小，排除
        }
        return l;                               // 第一个 >= target 的位置 = 插入点
    }
}
```

### 小结 / 语法点
- "插入位置 / 第一个 ≥ x" = lower_bound，背模板 B。
- 半开区间里 `r = mid`（不是 mid-1），循环条件 `l < r`。

## 74. 搜索二维矩阵（Search a 2D Matrix）· Medium

🔗 https://leetcode.cn/problems/search-a-2d-matrix/

### 题目大意
矩阵每行升序，且**每行第一个数大于上一行最后一个数**（整体可看作一维有序），判断 target 是否存在。

### 思路
**把二维当一维二分**。`m×n` 个元素整体有序，下标 `mid` 映射回二维坐标：行 `mid / n`，列 `mid % n`。直接套模板 A。

### 复杂度
- 时间 O(log(m·n))，空间 O(1)。

### 代码
```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length, n = matrix[0].length;
        int l = 0, r = m * n - 1;               // 把矩阵摊平成 [0, m*n-1]
        while (l <= r) {
            int mid = l + (r - l) / 2;
            int val = matrix[mid / n][mid % n]; // 一维下标 → 二维坐标
            if (val == target) return true;
            else if (val < target) l = mid + 1;
            else r = mid - 1;
        }
        return false;
    }
}
```

### 小结 / 语法点
- **一维下标 ↔ 二维坐标**：`row = idx / n`，`col = idx % n`。
- 注意区分 240 题（每行每列有序但行间无整体序）用的是右上角走法，本题整体有序可直接二分。

## 34. 在排序数组中查找元素的第一个和最后一个位置（Find First and Last Position）· Medium

🔗 https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/

### 题目大意
在升序数组中找 target 出现的**起止下标**；不存在返回 `[-1, -1]`。要求 O(log n)。

### 思路
**两次 lower_bound**：
- 起点 = 第一个 ≥ target 的位置。若越界或该位置不是 target，说明不存在。
- 终点 = （第一个 ≥ target+1 的位置）- 1，即最后一个等于 target 的位置。

### 复杂度
- 时间 O(log n)，空间 O(1)。

### 代码
```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        int left = lowerBound(nums, target);          // 第一个 >= target
        // 不存在：越界，或该位置不是 target
        if (left == nums.length || nums[left] != target) return new int[]{-1, -1};
        int right = lowerBound(nums, target + 1) - 1;  // 最后一个 == target
        return new int[]{left, right};
    }
    // 第一个 >= target 的下标（模板 B）
    private int lowerBound(int[] nums, int target) {
        int l = 0, r = nums.length;
        while (l < r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] >= target) r = mid;
            else l = mid + 1;
        }
        return l;
    }
}
```

### 小结 / 语法点
- **找区间 = 两次 lower_bound**：`[lower(t), lower(t+1)-1]`。比写两套不同二分更不易错。
- 用 `target + 1` 巧妙转化为"最后一个等于"。

## 33. 搜索旋转排序数组（Search in Rotated Sorted Array）· Medium

🔗 https://leetcode.cn/problems/search-in-rotated-sorted-array/

### 题目大意
升序数组在某点旋转后（如 `[4,5,6,7,0,1,2]`）查找 target，返回下标或 -1。要求 O(log n)。

### 思路
**判断哪半有序，再决定去哪半**。每次 `mid` 把数组分两段，**至少有一段是有序的**：
- 若 `nums[l] <= nums[mid]`，左半 `[l, mid]` 有序：若 target 落在 `[nums[l], nums[mid])` 内就去左半，否则去右半。
- 否则右半 `[mid, r]` 有序：若 target 落在 `(nums[mid], nums[r]]` 内就去右半，否则去左半。

### 复杂度
- 时间 O(log n)，空间 O(1)。

### 代码
```java
class Solution {
    public int search(int[] nums, int target) {
        int l = 0, r = nums.length - 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] == target) return mid;

            if (nums[l] <= nums[mid]) {                 // 左半有序
                if (nums[l] <= target && target < nums[mid]) r = mid - 1; // 在左半
                else l = mid + 1;
            } else {                                    // 右半有序
                if (nums[mid] < target && target <= nums[r]) l = mid + 1; // 在右半
                else r = mid - 1;
            }
        }
        return -1;
    }
}
```

### 小结 / 语法点
- 核心：**`nums[l] <= nums[mid]` 判断左半是否有序**，在有序半里用普通范围判断决定方向。
- 边界用 `<=`、`<` 要和闭区间配套，多画几个例子验证。

## 153. 寻找旋转排序数组中的最小值（Find Minimum in Rotated Sorted Array）· Medium

🔗 https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/

### 题目大意
旋转后的升序数组（无重复），找最小值。要求 O(log n)。

### 思路
**和右端点比较**。最小值是"旋转点"。比较 `nums[mid]` 与 `nums[r]`：
- `nums[mid] > nums[r]`：mid 在左边那段较大值里，最小值在 **mid 右侧**，`l = mid + 1`。
- `nums[mid] <= nums[r]`：mid 可能就是最小或在其左侧，`r = mid`（保留 mid）。
循环到 `l == r` 即最小值位置。（注意是和 `r` 比，不是和 `l` 比——和 l 比无法区分。）

### 复杂度
- 时间 O(log n)，空间 O(1)。

### 代码
```java
class Solution {
    public int findMin(int[] nums) {
        int l = 0, r = nums.length - 1;
        while (l < r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] > nums[r]) l = mid + 1; // 最小在右半（不含 mid）
            else r = mid;                          // 最小在左半（含 mid）
        }
        return nums[l];                            // l == r，即最小值
    }
}
```

### 小结 / 语法点
- **与右端点 `nums[r]` 比较**是这类"找旋转点"题的关键，比和左端比更可靠。
- `r = mid`（保留），`l = mid + 1`（排除），配 `l < r`。

## 4. 寻找两个正序数组的中位数（Median of Two Sorted Arrays）· Hard

🔗 https://leetcode.cn/problems/median-of-two-sorted-arrays/

### 题目大意
两个正序数组，返回合并后的中位数。要求 **O(log(m+n))**。

### 思路
转化为"**求两个有序数组合并后的第 k 小元素**"，再二分式地排除。
求第 k 小：比较两数组各自的第 `k/2` 个元素，**较小的那组的前 k/2 个一定不是第 k 小**，整体排除，k 相应减少，缩小问题规模。
- 总长度奇数：中位数 = 第 `(total/2 + 1)` 小。
- 偶数：中位数 = 第 `total/2` 小与第 `total/2 + 1` 小的平均。

### 复杂度
- 时间 O(log(m+n))，空间 O(1)。

### 代码
```java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int total = nums1.length + nums2.length;
        if (total % 2 == 1) {
            return findKth(nums1, nums2, total / 2 + 1);            // 奇数：正中间
        } else {
            double a = findKth(nums1, nums2, total / 2);           // 偶数：中间两个的平均
            double b = findKth(nums1, nums2, total / 2 + 1);
            return (a + b) / 2.0;
        }
    }

    // 求两个有序数组合并后的第 k 小（k 从 1 开始）
    private int findKth(int[] nums1, int[] nums2, int k) {
        int i = 0, j = 0;                       // 两数组当前起始下标
        while (true) {
            if (i == nums1.length) return nums2[j + k - 1]; // nums1 耗尽
            if (j == nums2.length) return nums1[i + k - 1]; // nums2 耗尽
            if (k == 1) return Math.min(nums1[i], nums2[j]);// 取较小的那个

            int half = k / 2;
            // 各取第 half 个元素的下标（不越界）
            int ni = Math.min(i + half, nums1.length) - 1;
            int nj = Math.min(j + half, nums2.length) - 1;

            if (nums1[ni] <= nums2[nj]) {
                k -= (ni - i + 1);              // 排除 nums1[i..ni]
                i = ni + 1;
            } else {
                k -= (nj - j + 1);              // 排除 nums2[j..nj]
                j = nj + 1;
            }
        }
    }
}
```

### 小结 / 语法点
- 难题。核心是把"中位数"化为"**第 k 小**"，再用"每轮排除 k/2 个"做到对数复杂度。
- 三个边界一定要先判：某数组空、k==1、取第 half 个时防越界（用 `Math.min` 截断）。
- 这题能默写出来面试加分；实在记不住，O(m+n) 的双指针归并也能过（但不满足进阶要求）。
