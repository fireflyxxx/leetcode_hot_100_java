## 📚 如何使用这套笔记

1. **先读基础两份**（不要跳）：
   - [00-Java语法速查.md](00-Java语法速查.md) —— 刷题用得到的 Java 语法/集合 API/常见坑，当字典查。
   - [01-做题套路总纲.md](01-做题套路总纲.md) —— 每类算法的"识别信号 + 通用模板"，先学会认题型。
2. **按板块刷题**：每个板块一个文件，顺序就是 Hot100 官方学习计划的顺序。
3. **每题的正确顺序**：先看"题目大意"自己想 5 分钟 → 卡住再看"思路" → 然后**自己默写代码** → 最后对照题解和"小结"。
4. 代码都是 **LeetCode 核心代码模式**（只写 `Solution` 类），可直接粘到 LeetCode 提交。

## 🎯 建议刷题节奏

- **第一遍**（建立手感）：按板块顺序刷，每题先想后看，重点是"识别题型 + 套模板"。
- **第二遍**（查漏补缺）：只刷第一遍卡住或不熟的题，盖住题解默写。
- **高频重点**：链表、二叉树、动态规划三块题量大、考点密，多花时间。
- 每道题做完回到 [01-做题套路总纲.md](01-做题套路总纲.md) 对一下它属于哪个套路，强化"信号→套路"的条件反射。
