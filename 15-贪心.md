## 121. 买卖股票的最佳时机（Best Time to Buy and Sell Stock）· Easy

🔗 https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

### 题目大意
只能买卖一次（先买后卖），求最大利润；无法盈利返回 0。
例：`[7,1,5,3,6,4]` → 第 2 天买(1)第 5 天卖(6)，利润 5。

### 思路
**一次遍历 + 维护最低价**。在第 i 天卖出的最大利润 = `prices[i] - 之前的最低价`。所以边遍历边记录"到目前为止的最低买入价 `minPrice`"，并用"今天价 - minPrice"更新最大利润。

### 复杂度
- 时间 O(n)，空间 O(1)。

### 代码
```java
class Solution {
    public int maxProfit(int[] prices) {
        int minPrice = Integer.MAX_VALUE; // 到目前为止的最低买入价
        int maxProfit = 0;
        for (int p : prices) {
            minPrice = Math.min(minPrice, p);          // 更新最低买入
            maxProfit = Math.max(maxProfit, p - minPrice); // 今天卖的最大利润
        }
        return maxProfit;
    }
}
```

### 小结 / 语法点
- 核心：**只需记住"此前最低价"**，无需回看整段。
- 既可看作贪心，也可看作"维护前缀最小值"的 DP。

## 55. 跳跃游戏（Jump Game）· Medium

🔗 https://leetcode.cn/problems/jump-game/

### 题目大意
`nums[i]` 表示从 i 能跳的最大步数，判断能否从下标 0 跳到最后一个下标。

### 思路
**维护能到达的最远位置 `maxReach`**。遍历，若当前下标 `i` 已经超过了 `maxReach`（说明跳不到 i），直接失败；否则用 `i + nums[i]` 更新最远可达。能走完全程即成功。

### 复杂度
- 时间 O(n)，空间 O(1)。

### 代码
```java
class Solution {
    public boolean canJump(int[] nums) {
        int maxReach = 0;                  // 当前能跳到的最远下标
        for (int i = 0; i < nums.length; i++) {
            if (i > maxReach) return false;        // 当前位置都到不了，失败
            maxReach = Math.max(maxReach, i + nums[i]); // 更新最远可达
        }
        return true;
    }
}
```

### 小结 / 语法点
- 贪心核心量：**"能到达的最远位置"**。一旦 `i > maxReach` 就断了。
- 不用关心怎么跳，只关心"够不够得着"。

## 45. 跳跃游戏 II（Jump Game II）· Medium

🔗 https://leetcode.cn/problems/jump-game-ii/

### 题目大意
保证能到终点，求到达最后一个下标的**最少跳跃次数**。

### 思路
**贪心分层（类 BFS）**。维护当前这一跳能覆盖的边界 `curEnd` 和遍历中见过的最远 `farthest`。当走到 `curEnd` 时，意味着当前这一跳的范围用完了，必须再跳一次，并把边界更新到 `farthest`。
形象理解：把能一步到的位置看成"第 1 层"，再下一步能到的是"第 2 层"……层数就是跳跃次数。

### 复杂度
- 时间 O(n)，空间 O(1)。

### 代码
```java
class Solution {
    public int jump(int[] nums) {
        int jumps = 0;       // 跳跃次数
        int curEnd = 0;      // 当前这一跳能到达的右边界
        int farthest = 0;    // 在当前边界内能探到的最远位置
        // 注意只遍历到 n-2：到达最后一个下标就不用再跳了
        for (int i = 0; i < nums.length - 1; i++) {
            farthest = Math.max(farthest, i + nums[i]);
            if (i == curEnd) {       // 走到当前边界，必须再跳一次
                jumps++;
                curEnd = farthest;   // 边界扩展到这一层能到的最远
            }
        }
        return jumps;
    }
}
```

### 小结 / 语法点
- 关键三量：`jumps / curEnd / farthest`。到达 `curEnd` 就 `jumps++` 并扩边界。
- 循环到 `n - 2`：避免在终点处多算一跳。

## 763. 划分字母区间（Partition Labels）· Medium

🔗 https://leetcode.cn/problems/partition-labels/

### 题目大意
把字符串划分成尽可能多的片段，使**同一字母只出现在一个片段**里，返回各片段长度。
例：`"ababcbacadefegdehijhklij"` → `[9,7,8]`。

### 思路
**记录每个字母最后出现的位置 + 贪心扩边界**。
- 先求出每个字母最后出现的下标 `last[c]`。
- 遍历，维护当前片段必须延伸到的终点 `end = max(end, last[当前字母])`。当遍历下标 `i == end` 时，说明当前片段内所有字母都不会再出现在后面了，可以切割。

### 复杂度
- 时间 O(n)，空间 O(1)（26 个字母）。

### 代码
```java
class Solution {
    public List<Integer> partitionLabels(String s) {
        int[] last = new int[26];
        for (int i = 0; i < s.length(); i++)
            last[s.charAt(i) - 'a'] = i;        // 每个字母最后出现的位置

        List<Integer> res = new ArrayList<>();
        int start = 0, end = 0;
        for (int i = 0; i < s.length(); i++) {
            end = Math.max(end, last[s.charAt(i) - 'a']); // 片段必须延伸到的终点
            if (i == end) {                     // 当前片段内字母都不会再出现 → 切
                res.add(end - start + 1);
                start = i + 1;
            }
        }
        return res;
    }
}
```

### 小结 / 语法点
- 贪心核心：**当前片段的终点 = 片段内所有字母 last 位置的最大值**。
- 与合并区间、跳跃游戏同属"维护一个动态边界"的贪心套路。
