## 3. 无重复字符的最长子串（Longest Substring Without Repeating Characters）· Medium

🔗 https://leetcode.cn/problems/longest-substring-without-repeating-characters/

### 题目大意
找出字符串中**不含重复字符**的**最长连续子串**的长度。
例：`"abcabcbb"` → `"abc"`，返回 `3`；`"pwwkew"` → `"wke"`，返回 `3`。

### 思路
**不定长滑动窗口**。窗口 `[left, right]` 内始终保持"无重复字符"这个性质：
- 右指针 `right` 每次纳入一个新字符。
- 如果新字符已经在窗口里（重复了），就不断从左边移出字符、`left++`，直到窗口里不再有这个重复字符。
- 每次窗口合法时，用 `right - left + 1` 更新最长长度。

用一个 `HashSet` 维护"窗口内有哪些字符"，实现 O(1) 的"在不在"判断。

形象理解：窗口像一条可伸缩的虫子，头（right）一直往前拱，遇到重复就把尾（left）往前收，始终保持身体里没有重复字符，记录最长身长。

### 复杂度
- 时间 O(n)：每个字符最多被 right 加入一次、被 left 移出一次。
- 空间 O(min(n, 字符集大小))。

### 代码（HashSet 版，最易懂）
```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        Set<Character> window = new HashSet<>(); // 当前窗口内的字符集合
        int left = 0, ans = 0;

        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);

            // 若新字符已在窗口里，从左侧不断收缩，直到把那个重复字符移出
            while (window.contains(c)) {
                window.remove(s.charAt(left));
                left++;
            }

            window.add(c);                       // 纳入新字符，窗口此时合法
            ans = Math.max(ans, right - left + 1); // 更新最长长度
        }
        return ans;
    }
}
```

### 代码（数组优化版，记录上次位置，left 直接跳）
```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int[] last = new int[128];          // 记录每个字符上次出现的下标+1（ASCII）
        int left = 0, ans = 0;
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            // 如果该字符上次出现位置在窗口内，left 直接跳到它的下一位
            left = Math.max(left, last[c]);
            ans = Math.max(ans, right - left + 1);
            last[c] = right + 1;            // 更新该字符最新位置（+1 便于 left 取值）
        }
        return ans;
    }
}
```

### 小结 / 语法点
- 这是"求最长"的不定长滑窗：**收缩到合法后**再更新答案。
- HashSet 版直观；数组版用"上次出现位置"让 left 一步跳到位，常数更优。
- `s.charAt(i)` 取字符；窗口长度是 `right - left + 1`。

## 438. 找到字符串中所有字母异位词（Find All Anagrams in a String）· Medium

🔗 https://leetcode.cn/problems/find-all-anagrams-in-a-string/

### 题目大意
在字符串 `s` 中找出所有是 `p` 的**字母异位词**的子串，返回它们的**起始下标**。
例：`s="cbaebabacd", p="abc"` → `[0, 6]`（`"cba"`、`"bac"`）。

### 思路
异位词 = 字母种类和数量都相同、只是顺序不同。子串长度固定为 `p.length()`，所以用**定长滑动窗口**：
1. 用 `need[26]` 统计 `p` 中每个字母的数量。
2. 维护一个长度为 `m = p.length()` 的窗口，用 `window[26]` 统计窗口内字母数量。
3. 窗口每右移一格：右端字符计数 +1；若窗口超长，左端字符计数 -1，保持长度恰为 m。
4. 当窗口长度等于 m 且 `window` 与 `need` 两个计数数组**完全相等**时，说明是异位词，记录起始下标。

用"计数数组相等"判断异位词，比每次排序快。

### 复杂度
- 时间 O(n)：每个位置 O(1) 维护，`Arrays.equals` 比较 26 个数是常数。
- 空间 O(1)：两个固定大小 26 的数组。

### 代码
```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> res = new ArrayList<>();
        int n = s.length(), m = p.length();
        if (n < m) return res;                 // s 比 p 短，不可能有异位词

        int[] need = new int[26];              // p 的字母频次
        int[] window = new int[26];            // 当前窗口的字母频次
        for (char c : p.toCharArray()) need[c - 'a']++;

        for (int i = 0; i < n; i++) {
            window[s.charAt(i) - 'a']++;       // 右端字符进窗口

            // 窗口长度超过 m，移除最左端字符（下标 i-m），保持窗口长度 = m
            if (i >= m) {
                window[s.charAt(i - m) - 'a']--;
            }

            // 窗口刚好凑满 m 个字符（i 从 m-1 开始），比较两个频次数组
            if (i >= m - 1 && Arrays.equals(window, need)) {
                res.add(i - m + 1);            // 记录窗口起始下标
            }
        }
        return res;
    }
}
```

### 小结 / 语法点
- **定长滑窗模板**：右端先进，超长则左端出，长度达标就检查/更新答案。
- `Arrays.equals(a, b)` 比较两个数组内容是否相等——异位词判定的利器。
- 起始下标 = 当前右端 `i - m + 1`。
- 同款思路可解 567「字符串的排列」（只判存在，不收集全部）。
